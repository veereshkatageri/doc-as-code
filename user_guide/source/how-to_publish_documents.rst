How to publish documents?
+++++++++++++++++++++++++

This page is intended to guide you the following with the help of onboarding template.

#. Rendering to html format
#. Publishing documents to Confluence
#. Rendering to PDF

All three publishing mechanism will be supported by doc-as-code toolchain.

Default build target types
==========================

For the above three publishing mechanism, the rendering the reStructuredText to html is always the \
first build type. In order to improve the quality of the documentation, spell checks will be \
followed next.
